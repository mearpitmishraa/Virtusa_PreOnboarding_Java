Banking Management System
It is a console-based application developed using Core Java as part of the Virtusa Pre-Onboarding Program.
The project simulates real-world banking operations with role-based access for Customers and Admin.
It allows users to register, login, deposit, withdraw, transfer funds, and view transaction history.
The Admin panel enables monitoring of customers, total bank balance, and account management.
The system is built using OOP principles and Java Collections Framework (HashMap, ArrayList).
It follows a modular, menu-driven CLI architecture for clean and structured execution.
The project demonstrates strong fundamentals in Core Java, data handling, and logical problem-solving.
